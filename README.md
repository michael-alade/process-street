## Process Street Test
Upload videos to wistia using the blueimp file upload plugin.

### Technologies
- AngularJS
- Blueimp JQuery File Upload
- Karma
- Nodejs
- Jasmine
- Gulp task runner
- NPM

### How to install
To install the app, run the following commands individually
```
$ npm i -g gulp karma
$ npm i
```

### How to run
```
$ gulp app
```

### Run tests
```
$ gulp test
```
