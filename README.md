## Process Street Test
Upload videos to wistia using the blueimp file upload plugin.

### Technologies
- AngularJS
- Blueimp JQuery File Upload
- Karma
- Nodejs
- Jasmine
- NPM

### How to install
To install the app, run the following commands individually
```
$ npm i -g karma
$ npm i
```

### How to run
```
$ npm start
```

### Run tests
```
$ npm test
```
